A simple and clean success pop-up message created using HTML, CSS, and JavaScript.
This project demonstrates how to display a confirmation pop-up after a user action such as submitting a form.

📌 Features

Simple and responsive design

Animated success pop-up

Built using pure HTML, CSS, and JavaScript

Beginner-friendly project

Easy to integrate into any website

🖼 Preview

A success popup appears after the user submits the form with a message:

"Thank You! Your details have been successfully submitted."
